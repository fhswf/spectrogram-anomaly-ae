## MA Dataset

Dataset for anomaly detection based on vibration data

- Train set (5000 samples): normal data only  
- Test set (5000 samples): normal + anomalies (~0.9%) 

### Notes

- Filenames are anonymized
- Highly imbalanced test set (~0.9% anomalies)  
- Suitable for unsupervised / semi-supervised anomaly detection  


### Usage

1. Train on `train/`  
2. Evaluate on `test/` using `labels.csv`
